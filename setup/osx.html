[Tested using Mac OSX 10.7.4]

Mac OSX 10.7.4 already comes with

  • Bash shell
  • vi and nano text editors
  • Python 2.7
  • Python setuptools
  • sqlite3

All installation needs to be done as a root user or user with certain administrator privileges.

Install make

If you do not already have access to make from within your shell, you will need to install XCode (which is free)

  • Go to the Apple app store
  • Search for XCode
  • Click Free
  • Click Install App

Once XCode has installed

  • Click Applications
  • Click XCode
  • Select XCode => Preferences...
  • Click Downloads
  • Select Command Line Tools
  • Click Install

You will now be able to run make within your bash shell.

Install Python nose

Run

$ easy_install nose

And check

$ nosetests

---------------------------------------------------------------------- Ran 0 tests in 0.003s

OK

Install Mercurial

Download the Mac binary package from http://mercurial.selenic.com/.

Install EasyMercurial

Download the Mac binary package from https://code.soundsoftware.ac.uk/projects/easyhg/files.

Install Firefox

You can download Firefox from the Mozilla web site at http://www.mozilla.org/en-US/firefox/new/.
