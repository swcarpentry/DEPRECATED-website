For all boot camps

Install the Anaconda CE Python Distribution

  • Download and Install Anaconda CE from http://continuum.io/anacondace.html
  • Use all of the defaults for installation except before pressing Finish make sure to check the Make Anaconda the default Python box.

Install Msysgit for bash

  • Download and Install msysgit from http://code.google.com/p/msysgit/downloads/list?q=full+installer+official+git
  • Use all of the default options in the installer

Install the Software Carpentry Installer

  • Download the installer from https://raw.github.com/swcarpentry/boot-camps/master/setup/swc-windows-installer.py
  • If the file opens directly in the browser select File -> Save Page As to download it to your computer
  • Double click on the file to run it

For some boot camps

If your boot camp will be using the following software please install it by following the instructions

SQLite

Only follow one set of these instructions depending on how your bootcamp will use SQLite.

Using the Firefox addon

  • Start Firefox
  • Select Tools → Add-ons
  • In the search field, enter sqlite
  • Click the search icon
  • Double-click to select SQLite Manager 0.7.7
  • Click Install
  • Click restart now
  • When Firefox restarts, Select either Web Developer → SQLite Manager or Tools → SQLite Manager

Using sqlite3 from the command line

  • Download sqlite-shell-win32-x86-SOMENUMBERS.zip from http://www.sqlite.org/download.html
  • Unzip the file
  • Copy sqlite.exe into C:\Windows

Mercurial

  • Download and install Mercurial http://mercurial.selenic.com/downloads/
  • If your boot camp is using EasyMercurial download and install it from http://easyhg.org/download.html
