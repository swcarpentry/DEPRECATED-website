Firefox SQLite manager

The SQLite Manager add-on can be installed through the "Add-ons Manager" in Firefox.

  • Start Firefox
  • Select Tools => Add-ons
  • In the search field, enter sqlite
  • Click the search icon
  • Double-click to select SQLite Manager 0.7.7
  • Click Install
  • Click restart now
  • When Firefox restarts, Select Tools => SQLite Manager
