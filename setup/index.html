To complete the boot camp you will need a web browser, the Unix Bash shell, the Git version control system (or Subversion or Mercurial, if your instructor is using them instead), a code editor (though any plain text editor will work in a pinch), and either a scientific Python installation that includes the IPython Notebook, NumPy, and matplotlib, or a full-featured installation of R. You may also need a simple database manager called SQLite, GNU Make or other software specific to your boot camp.

This page explains how to install this software. Please make sure you install everything before coming to your boot camp so that teaching can begin right away. (Wireless networks usually can't cope with 20 people trying to download these packages at the same time...) If you run into any problems, please let your boot camp organizer know as soon as possible.

You do not need to install everything listed below, since some packages are alternatives to one another.
Your instructor will let you know exactly which packages will be used in the boot camp you are attending.

Using a Virtual Machine

The easiest way for us to provide you with a consistent, pre-configured environment is to give you a Linux virtual machine. You can use this virtual machine from Windows, OS X, or even another Linux install. Install VirtualBox and download this virtual machine image (alternate download). Load the VM into VirtualBox by doing Import Appliance and loading the .ova file.

Native Installation

Many people prefer to install software directly on their machine so that it is integrated with their usual desktop environment. Even if you plan to do this, please download VirtualBox and the virtual machine as a backup—it can be difficult to get everything set up correctly on older machines.)

For Windows users: The Software Carpentry Installer installs several of the packages described below. Save the file to your hard drive, then double-click on the file to run it.

The Unix Bash Shell

Linux Bash is usually the default shell, but if your Linux installation gives you something else you can run Bash by opening a terminal and typing bash.
Mac Bash is the default shell in Mac OS X.
Windows Follow these instructions to install Git Bash.

Git

Linux Git should already be installed on your machine. To check, type which git at the command line. If it is not found, install it via your distribution's package manager (e.g., apt-get).
Mac Use the Git installer (the faster and simpler option) or install XCode and the command line tools (from the Download preferences pane). Note that the XCode download is over a gigabyte...
Windows Follow these instructions to install Git Bash. (You may already have done this to install Bash.)

Editor

Sublime Text is an excellent editor that works on Mac, Windows, and most Linux distributions. It's not free, but it has an unlimited trial period. Alternatively:

Linux Kate and gedit are often pre-installed on Linux.
Mac We recommend Text Wrangler, or Text Mate 2.
Windows Notepad++ is a popular free code editor for Windows.

Python

We recommend the all-in-one scientific Python distributions Anaconda and Canopy. Anaconda and Canopy Express are free to everyone, Canopy is free to academics. Setting these packages up may require using the shell; if you aren't comfortable doing this, please download the installer corresponding to your operating system before arriving and your instructor will help you set it up.

SQLite

If you are using the SQLite Manager add-on for Firefox, it can be installed through the "Add-ons Manager":

  1. Start Firefox.
  2. Select Tools → Add-ons.
  3. In the search field, enter sqlite.
  4. Click the search icon.
  5. Double-click to select SQLite Manager 0.7.7.
  6. Click Install.
  7. Click restart now.
  8. When Firefox restarts, select Tools→SQLite Manager.
Otherwise, follow the instructions below:
Linux SQLite may be pre-installed on your distribution. To check, type which sqlite3 in a shell. If it is not present:
  1. Download the SQLite Precompiled Binaries for Linux.
  2. Unzip the file.
  3. Test the program using: 
    $ ./sqlite3 --version
Mac SQLite is pre-installed on Mac OS X. Use which sqlite3 to check that you have it.
Windows
  1. Download the SQLite Precompiled Binaries for Windows.
  2. Unzip the file.
  3. Copy sqlite.exe into C:\Windows.
  4. Test the program using: 
    $ ./sqlite3 --version

GNU Make

Linux Make is pre-installed on all Linux distributions.
Mac This should be pre-installed on Mac OS X. If which make does not find it, you will need to install XCode (which you may already have done to install Git). Once that is done:
  1. Click Applications.
  2. Click XCode.
  3. Select XCode→Preferences...
  4. Click Downloads.
  5. Select Command Line Tools.
  6. Click Install.
Windows Make is installed by the Software Carpentry Installer.

R

You need to install and test R, a free software environment for statistical computing and graphics, along with RStudio and a few add-on packages.

R Download and install R from CRAN, the Comprehensive R Archive Network. It is highly recommended that you install a precompiled binary distribution for your operating system—use the links up at the top of the CRAN page.
RStudio Install RStudio. Do whatever is appropriate for your OS to launch RStudio. You should get a window similar to this screenshot, but yours will be more boring because you haven't written any code or made any figures yet. Put your cursor in the pane labelled Console, which is where you interact with the live R process. Create a simple object with code like x <- 2 * 4 (followed by enter or return), then inspect the result by typing x (again followed by enter or return). If it displays 8, you are good to go.

R is an extensible system and many people share useful code they have developed as a package via CRAN and GitHub. To install a package, for example the plyr package for data aggregation, here is one way to do it in the R console (there are others): 

{r, eval=FALSE, results="hide"}
install.packages("plyr", dependencies = TRUE)

Another package you may wish to play around with is knitr, which facilitates the creation of dynamic reports.

Mercurial

Download and install Mercurial. If your boot camp is using the EasyMercurial GUI, download and install it as well.

Supporting Tools

Mac OS X

XCode XCode is a set of free developer tools for Mac OS X. To install it:
  1. Go to the Apple app store.
  2. Search for XCode.
  3. Click Free.
  4. Click Install App.
Note that the XCode download is over a gigabyte, so please do this before arriving at the boot camp.
