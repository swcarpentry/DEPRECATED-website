[Tested using Windows 7 Enterprise, SP1 2009]

For Windows we use Cygwin which provides for Linux/UNIX interaction but fully integrated with the Windows file-system (unlike, for example, a virtual machine).

Install Cygwin for bash, make, text editors, SQLite, Mercurial and Python

  • Download setup.exe from http://www.cygwin.com/
  • Right-click setup.exe and select Run as administrator
  • Cygwin Setup window appears
  • Click Next>
  • Choose Installation Type window appears
  • Select Install from Internet
  • Click Next>
  • Choose Installation Directory window appears
  • Click Browse... and select a Root Directory, or use the suggested default
  • For Install For, select All Users
  • Click Next>
  • Select Local Package Directory window appears
  • Click Browse... and select a Local Package Directory, or use the suggested default
  • Click Next>
  • Select Connection Type window appears
  • Select Direct Connection
  • Click Next>
  • Choose A Download Site window appears
  • Select a mirror site or add one closer to you. I wanted a UK one, so entered User URL: http://www.mirrorservice.org and clicked Add
  • Click Next>
  • If you have not installed Cygwin before a Setup Alert dialog may appear, so just click OK
  • Select Packages window appears

Now, finally, you can select the packages to install...

  • Click Database
  • Scroll to sqlite3
  • Click New column until it toggles to 3.7.3-1 (or, the latest version)
  • Click Devel
  • Scroll to make
  • Click New column until it toggles to 3.82.90.1 (or, the latest version)
  • Scroll to mercurial
  • Click New column until it toggles to 2.3.1-1 (or, the latest version)
  • Click Editors
  • Pick your favourite editor...
  • Scroll to nano
  • Click New column until it toggles to 2.2.5-1 (or, the latest version)
  • Scroll to vim
  • Click New column until it toggles to 7.3.646-1 (or, the latest version)
  • Scroll to xemacs
  • Click New column until it toggles to 21.4.22-1 (or, the latest version)
  • Click Python
  • Scroll to python
  • Click New column until it toggles to 2.6.8-2 (or, the latest Python 2.x version)
  • Click Shells
  • Scroll to bash
  • Check New column until it toggles to 4.1.10-4 (or, the latest version)
  • Click Next>
  • Resolving Dependencies window appears
  • Click Next>
  • Wait until it's all done!

Now, set up the path...

  • Find Computer on the desktop or in Windows Explorer
  • Right Click and select Properties
  • System window appears
  • Click Advanced system settings.
  • Enter your local administrator password
  • System properties window appears
  • Click Advanced tab
  • Click Environment Variables...
  • In the System variables list, select Path
  • Click Edit...
  • Edit System Variable dialog appears
  • Add ;C:\cygwin\bin to the end of the Variable value.
  • Click OK to close Edit System Variable
  • Click OK to close Environment Variables
  • Click OK to close System properties

Install Python setuptools

  • Visit http://pypi.python.org/pypi/setuptools and download the egg file corresponding to your Python version. Download a standard egg and not the Windows installer. For example, I downloaded setuptools-0.6c11-py2.6.egg
  • Save your egg into your Cygwin home directory, for example

    C:\MikeLocal\cygwin\home\mjj
  • Right-click Cygwin Terminal icon
  • Select Run as administrator
  • Enter your local administrator password.
  • Execute the egg,

    $ cd /home/mjj
    $ sh setuptools-0.6c9-py2.4.egg

Install Python nose

  • Right-click Cygwin Terminal icon
  • Select Run as administrator
  • Enter your local administrator password
  • At the prompt, run

    $ easy_install nose
  • And check

    $ nosetests
    ----------------------------------------------------------------------
    Ran 0 tests in 0.003s
    OK

Install EasyMercurial

Download the EasyMercurial Windows installer from http://easyhg.org/download.html.

Double-click on the installer e.g. EasyMercurial-1.2.2-Setup.msi.

Install Firefox

You can download Firefox from the Mozilla web site at http://www.mozilla.org/en-US/firefox/new/.
