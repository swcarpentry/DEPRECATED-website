[Tested using Ubuntu 11.04 running within VMWare]

Ubuntu 11.04 already comes with

  • Bash shell
  • Make
  • vi and nano text editors
  • Python 2.7
  • Firefox

All installation needs to be done as a root user or user with certain administrator privileges. If you have such access you can do

$ sudo su -

Then enter your usual password.

Install Emacs or XEmacs

Run

$ apt-get install emacs23
$ emacs --version

Run

$ apt-get install xemacs21
$ xemacs --version

Install Python setuptools

Run

$ apt-get install python-setuptools

Install Python nose

Run

$ easy_install nose

And check

$ nosetests
----------------------------------------------------------------------
Ran 0 tests in 0.003s
OK

Install Mercurial

Run

$ apt-get install mercurial
$ hg --version

Install EasyMercurial

Download the EasyMercurial Debian/Ubuntu .deb package from http://easyhg.org/download.html. This can be done at the command-line by copying the download link from the download page (using right-click and Copy Shortcut) then doing a wget e.g.

$ wget https://code.soundsoftware.ac.uk/attachments/download/487/easymercurial_1.2.2cc-1_i386.deb

Beware that this link may have been updated since this page was written, so check the download page above for the URL to use.

Run

$ dpkg -i easymercurial_1.2cc-1_i386.deb

This will throw up a lot of warnings about dependency problems. This is expected!

Now run

$ apt-get -f install
$ dpkg -i easymercurial_1.2cc-1_i386.deb

Go to the Applications => Programming menu and you should see an EasyMercurial command. Select it and EasyMercurial should start.

Install SQLite

SQLite does not need administrator privileges to run.

Download the SQLite Precompiled Binaries for Linux package from http://www.sqlite.org/download.html.

This can be done at the command-line by copying the download link from the download page (using right-click and Copy Shortcut) then doing a wget e.g.

$ wget http://www.sqlite.org/sqlite-shell-linux-x86-3071401.zip

Beware that this link may have been updated since this page was written, so check the download page above for the URL to use.

Unzip

$ unzip sqlite-shell-linux-x86-3071100.zip
$ ./sqlite3 --version

sqlite3 is the binary so you can put it and use it anywhere by anyone.
